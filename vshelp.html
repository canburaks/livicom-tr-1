Description
The Livi VS impact sensor (hereafter referred to as the sensor) is designed to detect displacement or impact that occur when glass, metal, brick and reinforced concrete structures are being cut, drilled or broken.

The sensor detects both single (e.g. impact, explosion) and periodic (e.g. drilling) pulses. The sensor also detects movements and angular displacement of the guarded surface.

The sensor switches to the alarm mode when it detects an impact on the surface (if the guard is enabled):

  1. the sensor indicator blinks red once,
  2. the sensor sends an alarm alert to the Livi Smart Hub (hereafter referred to as the hub).
Sensor appearance
  1. LED indicator
  2. Enclosure latch
  3. Protective film
  4. Battery
  5. Tamper button
Binding to the hub
The sensor must be unpacked and allowed to reach room temperature for at least two hours before handling if it was transported or stored at low temperatures.
Only the owner of the site can bind the sensor to the hub. Make sure that the site guard is disabled and the hub is online before binding.
Follow these steps to bind the sensor to the hub:

1. Pull the protective film out of the battery compartment. The sensor indicator will start blinking blue once the sensor is switched to the binding mode.

2. In the Livicom app, open the "Devices" screen. In the upper right corner of the screen tap "+" and select "Add Device".

The system compiles a list of radio devices available for binding within the range of the hub radio channel.

3. Select the sensor from the list and check that its name and serial number in the app match the data on the sensor enclosure.

4. Tap CONTINUE and follow on-screen instructions in the app to bind the sensor to the hub. The sensor indicator will blink green 5 times after successful binding.

You should set up the level of sensitivity for the sensor. By default the sensitivity level is set to 30 %.

You will be able to change the level of sensitivity later on the sensor settings screen (see below). Detection zone test can help you to define the optimal sensitivity level.

Note that if you set the sensitivity level to 0 %, then the sensor won't be disabled, but it will detect only strong vibration.

The sensor switches to the binding mode only for 60 seconds. If you have not bound it to the hub within this period, remove the battery from the sensor for 30 seconds, and reinstall it (observing polarity). The sensor will switch to binding mode again.
Removing or replacing the battery
  1. Open the sensor enclosure to remove or replace the battery: press one of the latches (2) on the short side of the enclosure using a flat-blade screwdriver,
  2. Pull the lid up while pressing the latch.

3. Remove the battery.

4. Install a new CR123A battery (observing polarity) if necessary and close the sensor enclosure.

Choosing a location for the sensor
The sensor is designed to be mounted on a guarded surface.
    DO NOT install the sensor outdoors, in places with high humidity, or at temperatures exceeding the operating temperature range (see "Specifications").
    Evaluating signal strength
    Check the quality of the connection between the sensor and the hub at the intended location of the sensor. There are two ways to evaluate the signal strength:

    1. In the Livicom app, on the sensor settings screen.
    2. With the help of the LED indication on the sensor. Double-click on the tamper button and look at the sensor indicator. Interpret the indication using the table below.

    Sensor installation
    1. Open the sensor enclosure: press one of the latches (2) on the short side of the enclosure using a flat-blade screwdriver and then pull the lid up while pressing the latch.
    2. Fasten the base of the sensor enclosure at the selected location using a supplied mounting kit.*
    3. Close the sensor enclosure.
      * We recommend fastening the sensor base with self-tapping screws, as duct tape can absorb vibration and so reduce the sensor sensitivity
      Checking the sensor operation
      Check the operation of the sensor after its installation.

      Enable full guard of the site through the Livicom app and wait for 5 minutes. Create vibration on the guarded surface (hit or move the surface). Make sure that the sensor indication matches the information in the table "LED indication" and you see an alarm alert in the app.

      Contact technical support (mail to: support@livicom.ru) if you see an incorrect indication or do not receive the alert.
      Attention

      The sensor does not monitor impacts when the guard is disabled, in order to extend battery life (in the Livicom app the sensor status will always be OK if the guard is disabled). The sensor requires from 3 to 5 minutes to switch to the standby mode and start monitoring impacts after the guard is enabled.

                Renaming the sensor
                Only the owner of the site can rename the sensor. Make sure that the site guard is disabled and the hub is online before renaming.
                • In the upper right corner of the settings screen tap on three dots ("Menu" button).
                • Select Rename in the menu.
                • Enter a new name and save it.

                Operational parameters
                • Radio signal strength between the sensor and the hub (no connection / poor / average / good):
                If there is no connection with the hub, check whether the hub is turned on. If the hub is off, turn the hub on and again check the signal strength on the sensor settings screen.
                Check whether there are any objects obstructing radio signal (for example, metal objects or mirrors) between the hub and the sensor. Change the sensor installation location so that radio signal could pass between devices without obstructions.
                Check that the sensor is installed within radio signal range of the hub. If the sensor is installed too far from the hub, move it to a new place where connection between the sensor and the hub will be restored.
                The location of the hub also affects the signal strength. In some cases even a slight displacement of the hub or even its rotation gives a significant improvement in a quality of its communication with devices.

                • Battery status (OK / low battery).
                If the battery runs out, you will receive the push notification. The notification will also be displayed in the event log in the app and on the sensor settings screen. Follow instructions to replace the battery.

                • Enclosure status (open / closed).
                Operational settings
                Only the owner of the site can change settings for the sensor. Make sure that the site guard is disabled and the hub is online before making changes.
                • Delay for entry / exit - slide the switch to the right if the sensor is mounted in the entrance area of the site AND the site guard is controlled by the Livi RFID security control panel.
                • Night guard - slide the switch to the right if the sensor should be switched to the guarding mode when the night guard is enabled.
                • Sensitivity - you can adjust the level of sensitivity for the sensor to avoid false alarms. By default the sensitivity level is set to 30 %. Detection zone test can help you to define the optimal sensitivity level and position for the sensor.
                  If you set the sensitivity level to 0 %, then the sensor won't be disabled, but it will detect only strong vibration.
                    • Detection zone test – this test is designed to help choosing the best position and sensitivity level for the sensor.
                    Detection zone test
                    Detection zone test helps to choose the best position and optimal sensivity for the sensor by testing the sensor at the intended location.

                    Open the sensor settings screen as described above and tap on the line Detection zone test.
                    Make sure that full guard of the site is disabled before starting the test. If the sensor is not in the main group, then disable also the guard of the group, to which the sensor belongs
                    Tap on START TEST and wait for 3-5 minutes in order to switch the sensor to the testing mode.

                    Perform strokes of various strengths all over the guarded surface. The sensor will transmit data about its current state every 1-2 seconds. Every time the sensors will detect an impact a vibration on smart phone will be triggered. If the sensor does not detect all impacts, then disrupt the test, increase the sensor sensitivity and start the test again.

                    By default the detection zone test lasts for 10 minutes. You can interrupt it any time before that by tapping COMPLETE.
                    To eliminate false alarms make sure that at the selected sensitivity level the sensor does not react to:

                    • The vibration from the movement of vehicles outside the guarded site.
                    • The vibration from other sources that are located in the detection zone of the sensor.

                    If necessary, adjust the sensitivity level of the sensor and test the detection zone again.
                    Maintenance
                    Keep the sensor free of dust and dirt. Replace the battery as soon as possible after you receive a low battery notification in the Livicom app.
                    Attention

                    Do not wipe the sensor with substances containing alcohol, acetone, gasoline and other active solvents
                    Deleting the sensor (unbinding from the hub)
                    Only the owner of the site can unbind the sensor from the hub. Make sure that the site guard is disabled and the hub is online before deleting the sensor.
                    There are two ways to unbind the sensor from the hub:

                    1. In the Livicom app, using the menu on the sensor settings screen.
                    2. Using the sensor tamper (5). If you need to forcefully delete the sensor, remove the battery from the sensor for 30 seconds, then press the tamper button and while holding it, reinstall the battery, observing polarity. Release the tamper button and quickly click on it until the sensor indicator starts blinking blue.

                    Warranty
                    The manufacturer LLC "NPP Stels" guarantees that the sensor meets AGNS.421453.001 TU technical requirements, provided that the consumer complies with the conditions of transportation, storage, installation and operation.

                    The warranty period is 5 years from the manufacturing date. The warranty does not apply to batteries.

                    The warranty does not cover the following cases:
                    1. Non-compliance with the intended operating conditions for the sensor;
                    2. Mechanical damage to the sensor;
                    3. Repairs to the sensor by a third party (a person or a company other than the Manufacturer).
